## 🌡️ Project Description
This is a console-based **Temperature Converter** developed in C++ as part of the CodTech Internship (Task CITS2826). The program converts temperature values between **Celsius**, **Fahrenheit**, and **Kelvin** through a simple menu-driven interface.

## ✨ Features
- 🔄 Celsius ↔ Fahrenheit
- 🔄 Celsius ↔ Kelvin
- 🔄 Fahrenheit ↔ Kelvin
- 🎯 Accurate results up to 2 decimal places
- 📋 Simple, menu-driven console interface

## 🛠️ Tech Stack
- **Language:** C++
- **Concepts used:** OOP (Classes & Objects), Functions, Switch-case, Formatted Output (`iomanip`)
